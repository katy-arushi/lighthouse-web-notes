  - Add collaborators
  - clone the repo

### Installation

- `npm install` to install the dependencies
- `copy the .env file`
- `npm run db:setup` to run the schema and the seeds
- `npm run local` to run the server with Nodemon

_Replace the current schema and seed files with your own_

## Routes
